# Buoy Desktop

**The desktop companion for [BUOY Devtools](https://github.com/Buoy-gg/buoy).**

See everything your React Native app is doing — from your desk. Buoy Desktop mirrors the on-device Buoy devtools to a native desktop dashboard in real time: network requests, storage, React Query cache, and more, streamed live from any connected device.

📚 **[Full Documentation](https://buoy.gg)** · 🚀 **[Get Buoy for your app](https://github.com/Buoy-gg/buoy)**

---

## What it does

- **Live device mirroring** — connect any device running [Buoy devtools](https://github.com/Buoy-gg/buoy) and watch its network, storage, and query state in real time
- **Remote actions** — edit storage values, refetch queries, simulate loading/error states, and clear caches on the device, from your desktop
- **Multi-device** — switch between connected simulators and physical devices (iOS, Android, web)
- **Same tools, bigger screen** — the exact Buoy tool UIs you know from the device, on your desktop and in your browser
